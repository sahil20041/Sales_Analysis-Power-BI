Sales Analysis Dashboard (Power BI)

Overview

An interactive Power BI dashboard for analyzing sales performance, customer behavior, product categories, and returns using DAX, Power Query, and data modeling.

Features

* KPI Cards (Sales, Orders, Customers, Quantity, Average Sales, Returns)
* Sales Trend Analysis
* Category & Segment Analysis
* Waterfall & Funnel Charts
* Matrix & Table Visuals
* Interactive Slicers
* Drill-through Navigation

Tools

* Power BI Desktop
* Power Query
* DAX

Dataset

* Orders
* Products
* Customers
* Returns

Skills Demonstrated

* Data Cleaning
* Data Modeling
* DAX Measures
* Dashboard Design
* Data Visualization
* Business Analysis
* Drill-through
